# /inbox — захват заметки

Корень brain (далее `$BRAIN`) — содержимое файла `~/.claude/brain-dir`; файла нет → `~/brain`.

Аппендит заметку в `$BRAIN/inbox/YYYY-MM-DD.md` (дата — сегодняшняя, локальное время). Ноль трения: никакой классификации, никаких вопросов, никаких правок остального brain.

## Шаги

1. Возьми текст заметки из аргументов вызова. Если текста нет — спроси одной строкой «что записать?».
2. Определи сегодняшнюю дату и время: `date '+%Y-%m-%d %H:%M'`.
3. Если файла `$BRAIN/inbox/<YYYY-MM-DD>.md` нет — создай с заголовком:

   ```markdown
   # Inbox <YYYY-MM-DD>
   ```

4. Допиши в конец файла:

   ```markdown
   - **HH:MM** — <текст заметки как есть>
   ```

5. Ответь одной строкой: `inbox: записано в <YYYY-MM-DD>.md`. Не пересказывай заметку, не предлагай её обработать.

## Правила

- Текст заметки сохраняется **как есть** — не редактировать, не сокращать, не переводить.
- НЕ классифицировать, НЕ маршрутизировать, НЕ трогать INDEX.md и другие файлы brain — это работа `/inbox-process`.
- Заметка содержит секрет (токен, пароль) → заменить секрет на `[secret removed]` и сказать об этом пользователю.
