# Внешние ссылки — не угадывать

Не угадывай и не реконструируй URL по шаблону для внешних систем: личных кабинетов, админок, вики, дашбордов и т.п. Даже если паттерн выглядит «каноническим» — он может быть устаревшим или не тем, который нужен пользователю.

## Why

Угаданный URL в документе — мина замедленного действия: выглядит достоверно, а обнаружит подмену только читатель, когда перейдёт. Две похожие системы часто имеют разные ветки URL, и «логичная» реконструкция попадает не туда.

## How to apply

- Нужно сослаться на страницу / документ / кабинет — либо спроси URL у пользователя, либо найди точную ссылку в существующих записях brain.
- Без запроса можно использовать только ссылки, которые уже встречались в этом brain или в этой сессии (скопированы as-is, не реконструированы).
