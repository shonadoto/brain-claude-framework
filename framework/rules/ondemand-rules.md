# Правила по требованию — прочитать ПРИ ВХОДЕ в тему, до первого действия

Полные тексты — `__BRAIN_DIR__/rules-ondemand/<файл>`. Здесь — триггер и ядро правила. Вход в тему из левой колонки → прочитай файл(ы) правой ДО первого действия в теме, не после ошибки. Ядро — напоминание, не замена полного текста.

Зачем этот слой: правила с узким триггером («когда делаю X») не должны раздувать автозагрузку каждой сессии. Сюда их выносит пользователь или `/retro` (с одобрения). При добавлении файла в `rules-ondemand/` — строка в этой таблице обязательна, иначе правило не найдут.

| Вход в тему (триггер) | Ядро | Полный текст (rules-ondemand/) |
|---|---|---|
| (пока пусто — таблица заполняется по мере появления правил) | | |

Пример строки:

```
| Готовлю презентацию для клиента | сначала структура на одобрение, потом слайды; цифры только из проверенных источников | <имя-файла>.md |
```
