# Proposals — предложения по улучшению

Паттерны, достигшие порога ≥3 наблюдений, превращаются здесь в предложения: новое правило, скилл, рецепт. Внедряется **только после одобрения пользователя** (`/retro` показывает и спрашивает).

Формат: что наблюдалось (со ссылками) → что предлагается → цена внедрения → статус (ждёт решения / одобрено / отклонено (причина) / внедрено).
