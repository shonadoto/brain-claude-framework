# Паттерны — наблюдения со счётчиками

Пополняется скиллами `/retro` и `/digest`. Паттерн = повторяющееся наблюдение (ручная цепочка действий, повторный вопрос, грабли) со ссылками на конкретные сессии. При ≥3 наблюдениях — кандидат в `proposals.md`. Наблюдение без ссылки на конкретную сессию не записывается.

Формат записи:

```markdown
## <имя-паттерна>
- суть: <что повторяется>
- наблюдения: N
  - YYYY-MM-DD, сессия <id>: <коротко>
```
